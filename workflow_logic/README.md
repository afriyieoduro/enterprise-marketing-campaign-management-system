# Workflow Logic

This folder contains notes related to the workflow and lifecycle logic of the Marketing Campaign Management Automation System.

## Campaign Lifecycle

- Draft
- Pending Approval
- Approved
- Rejected
- Scheduled
- Sent
- Cancelled

## Workflow Goals

- prevent unauthorized sending
- ensure campaigns follow an approval path
- support scheduling for future release
- maintain status visibility
- improve coordination within the marketing process

## Workflow Actions

- create campaign
- edit campaign
- submit for approval
- approve
- reject
- schedule
- send test
- send live campaign
- cancel or stop workflow when required
